Para que serve uma amostra de tamanho grande?¶

Uma vez que as observaes so independentes (independents) e a distribuio da populao (population) destas observaes no muito enviesada (skewed), ento, o fato de termos amostras aleatrias (random samples) grandes garante que:

  1. Teremos uma distribuio amostral (sampling distribution) da mdia que aproximadamente Normal (como j foi visto aqui);
  2. A estimativa para o erro padro (standard error) $SE$ desta distribuio amostral (sampling distribution) ser confivel.

Ento, para verificarmos o ponto 2, considere que a distribuio da populao (population) seja dada por

$$ \textrm{pop} \sim N \left( \mu=0, \sigma=1 \right) $$

ou seja, uma populao (population) com mdia $\mu = 0$ e desvio padro $\sigma = 1$.

Considere que iremos coletar $m$ amostras aleatrias (random samples) desta populao (population) onde $m=$ sampling_distribution_size. Considere tambm que cada uma destas $m$ amostras aleatrias (random samples) tenha um tamanho $n$, onde $n=$ random_sample_size.

(Inspirao: [Inferential Statistics](https://www.coursera.org/learn/inferential-statistics-intro/home/welcome), by Duke University, at Coursera)

Se calcularmos a mdia de cada uma destas $m$ amostras aleatrias (random samples), ento, teremos uma distribuio amostral (sampling distribution) das mdias, denotada no cdigo por sampling_distribution_of_means.

A partir da distribuio amostral (sampling distribution) das mdias ns podemos:

  1. Estimar um valor $\hat{\mu}$ (estimated_mean) para a mdia $\mu$ da populao (population);
  2. Estimar o grau de preciso $SE$ (standard_error_sampling_dist) com o qual este valor estimado $\hat{\mu}$ (estimated_mean) representa a mdia $\mu$ da populao (population).

Esta estimativa do grau de preciso dada pelo erro padro (standard error), cuja definio pela wikidepia a seguinte:

The standard error (SE) of a statistic (usually an estimate of a parameter) is the standard deviation of its sampling distribution or an estimate of that standard deviation. If the parameter or the statistic is the mean, it is called the standard error of the mean (SEM) (Fonte: https://en.wikipedia.org/wiki/Standard_error)

Primeiramente, o que queremos verificar aqui se a preciso dada pelo erro padro (standard error) aumenta na medida em que o tamanho $n$ das amostras aleatrias (random samples) tambm aumenta.

import numpy as np

random_sample_sizes = [10, 30, 100, 500, 1000, 5000, 10000, 50000, 100000]
sampling_distribution_size = 1000
estimated_means = []
standard_errors = []
for random_sample_size in random_sample_sizes:
    sampling_distribution_of_means = []
    for _ in range(sampling_distribution_size):
        random_sample = pop_distribution.rvs(size=random_sample_size)
        sampling_distribution_of_means.append(np.mean(random_sample))

    estimated_mean = np.mean(sampling_distribution_of_means)
    estimated_means.append(estimated_mean)
    
    standard_error_sampling_dist = np.std(sampling_distribution_of_means)
    standard_errors.append(standard_error_sampling_dist)

import pandas as pd
import altair as alt
alt.renderers.enable('notebook') # for the notebook only (not for JupyterLab) run this command once per session

def plot_truemean_estimatedmean_moe(true_mean, estimated_means, estimated_standard_errors, random_sample_sizes):
    data = pd.DataFrame({
        'true_mean': true_mean, 'random_sample_size': random_sample_sizes, 
        'estimated_mean': estimated_means, 'estimated_se': estimated_standard_errors
    })
    
    true_mean_plot = alt.Chart(data).mark_rule(color='red').encode(
        y='true_mean:Q',
        size=alt.value(1)
    )
    
    estimated_mean_plot = alt.Chart(data).mark_line().encode(
        x='random_sample_size:O',
        y='estimated_mean'
    )
    
    margin_of_error_plot = alt.Chart(data).mark_area(opacity=0.3).encode(
        x='random_sample_size:O',
        y='mu_hat_sup:Q',
        y2='mu_hat_inf:Q'
    ).transform_calculate(
        mu_hat_sup='datum.estimated_mean + (1.96*datum.estimated_se)',
        mu_hat_inf='datum.estimated_mean - (1.96*datum.estimated_se)',
    )
    
    chart1 = (estimated_mean_plot + margin_of_error_plot).properties(title='Estimated Mean +- 1.96*SE')
    chart2 = (true_mean_plot + margin_of_error_plot).properties(title='True mean (in red)')
    chart3 = (true_mean_plot + estimated_mean_plot).properties(title='True (red) and Estimated (blue) means')
    
    return chart1 | chart2 | chart3

Logo,  possvel perceber que na medida em que o tamanho $n$ (`random_sample_size`) das **amostras aleatrias** (***random samples***) aumenta, o ***grau de preciso*** $SE$ da estimativa $\hat{\mu}$ tambm aumenta (ou seja, a **margem de erro** reduz, reduzindo tambm a largura do **intervalo de confiana**). 




Alm disto, pelo Teorema Central do Limite (Central Limit Theorem - CLT) temos que: a distribuio de estatsticas de uma amostra (sample statistics)  aproximadamente normal (nearly normal), centrada na mdia $\mu$ da populao (population), e com um desvio padro (standard deviation) igual ao desvio padro (standard deviation) da populao (population) dividido pela raiz quadrada do tamanho $n$ da amostra.

$$
\hat{\mu} \sim N\left(mean=\mu, SE=\frac{\sigma}{\sqrt{n}}\right)
$$
No entanto, nem sempre temos o desvio padro (standard deviation) $\sigma$ da populao (population). Nestes casos, tipicamente, o desvio padro (standard deviation) $\sigma$ da populao (population)  estimado como sendo o desvio padro $s$ (standard deviation) da amostra aleatria (random sample) em questo. Ento, uma estimativa para o erro padro (standard error) da distribuio amostral (sampling distribution) a partir de uma amostra aleatria (random sample) de tamanho $n$  dada por:

$$
\hat{SE} = \frac{s}{\sqrt{n}}
$$
O que queremos verificar aqui  se esta estimativa $\hat{SE}$ para o erro padro (standard error) fica mais precisa na medida em que o tamanho $n$ das amostras aleatrias (random samples) aumenta.

def estimate_SEs(pop_distribution, random_sample_sizes):
    estimated_standard_errors = []
    for random_sample_size in random_sample_sizes:
        random_sample = pop_distribution.rvs(size=random_sample_size)
        estimated_standard_error = np.std(random_sample) / np.sqrt(random_sample_size)
        estimated_standard_errors.append(estimated_standard_error)
    return estimated_standard_errors

estimated_standard_errors1 = estimate_SEs(pop_distribution, random_sample_sizes)
estimated_standard_errors2 = estimate_SEs(pop_distribution, random_sample_sizes)
